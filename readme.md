# Eclipse Addon Store — 8SPINE Source

An 8SPINE module source that bridges the [Eclipse Music](https://eclipsemusic.app) community addon ecosystem.

## What it includes

- **All in One** (default) — HiFi lossless, SoundCloud, Podcasts, Radio, Internet Archive
- Auto-falls back through the full Eclipse addon registry

## How to install in 8SPINE

1. Go to **8SPINE → Sources → Add Source**
2. Paste the URL below and tap **Add Source**

```
https://<your-github-username>.github.io/<this-repo-name>/index.json
```

## File structure

```
index.json                        ← source registry (paste this URL into 8SPINE)
modules/
  eclipseAddonStore.8spine        ← module code
```
