# Eclipse → 8SPINE Bridge

A serverless bridge that syncs [Eclipse Music](https://eclipsemusic.app) addons into [8SPINE](https://8spine.com) as a live module source. Automatically fetches the latest Eclipse registry and converts addons into installable 8SPINE modules on-the-fly.

**Source URL:** `https://eclipse-8spine.vercel.app/index.json`
